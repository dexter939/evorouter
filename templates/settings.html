API Documentation

The router provides a REST API for remote management. To use the API:

  1. Create an API client and note the API key
  2. Include the API key in the X-API-Key header with each request
  3. Use the following endpoints:
    • /api/v1/system/info - Get system information
    • /api/v1/network/interfaces - List network interfaces
    • /api/v1/network/interface/{id} - Update interface settings
    • /api/v1/freeswitch/status - Get FreeSWITCH status
    • /api/v1/freeswitch/extensions - List FreeSWITCH extensions
    • /api/v1/logs - Get system logs
